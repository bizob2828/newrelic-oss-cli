# Usage
<!-- usage -->
```sh-session
$ npm install -g gh-manager
$ ghm COMMAND
running command...
$ ghm (-v|--version|version)
gh-manager/0.0.0 darwin-x64 node-v10.16.3
$ ghm --help [COMMAND]
USAGE
  $ ghm COMMAND
...
```
